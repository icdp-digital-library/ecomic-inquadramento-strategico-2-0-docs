# Ecomic - ecosistema digitale per la cultura, Documento di inquadramento strategico

Il Ministero della Cultura, la Direzione generale Digitalizzazione e comunicazione e l’Istituto centrale per la digitalizzazione del patrimonio culturale – Digital Library hanno pubblicato la **versione 2.0** del documento di inquadramento strategico **Ecomic – ecosistema digitale per la cultura**.
